# Tests

Any changes to scripts in this folder must be reflected in this file.

## Stats
`./stats.sh` — prints char count, word count, unique characters, and printable ASCII coverage (out of 95) for each test case in `test_cases.py`. No output files, just logs.

Output format:
```
Name                                Chars    Words  Unique  Coverage
--------------------------------------------------------------------
tiny_note                             141       25    34/95    36.2%
...
--------------------------------------------------------------------
Total: 22 test cases
```
